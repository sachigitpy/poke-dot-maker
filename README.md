# ぽこポケ文字ドットメーカー (24×24版)

ぽこあポケモンで使用する、モニター用文字ドットの設計図生成ツールです。
ブラウザ上で文字を入力するだけで、24×24のドット配置を確認できます。

## 🚀 ツールURL
[https://sachigitpy.github.io/poke-dot-maker/]

## 🛠 使い方
1. 上記のURLを開きます。
2. **「文字入力」** 欄に、作りたい文字（漢字・ひらがな等）を1文字入力します。
3. **「設計図を生成」** ボタンを押すと、下のグリッドにドットが表示されます。
4. マス目をクリックすると**緑色（完了マーク）**に変わります。建築の進捗管理に使ってください。

## 📏 仕様
- **全体キャンバス**: 50 × 30 マス
- **文字描画エリア**: 中央 24 × 24 マス（赤枠部分）
- **座標表示**: マス目にマウスを乗せると、左上に座標（X, Y）が表示されます。
